# Session Stop Checklist

Use this checklist before ending a long-running coding session.

## Required
- [ ] update `docs/agent-memory/SESSION_SUMMARY.md`
- [ ] update `docs/agent-memory/TASK_BOARD.md`
- [ ] append to `docs/agent-memory/DECISION_LOG.md` if a real decision/root cause was confirmed
- [ ] update `docs/agent-memory/HANDOFF.md`

## Handoff Must Include
- [ ] current focus
- [ ] verified findings
- [ ] open problems
- [ ] exact files/specs to inspect next
- [ ] recommended next action

## Validation Rules
- [ ] no vague phrases like "worked on several things"
- [ ] no transcript dump
- [ ] assumptions are marked as assumptions
- [ ] stable facts are not silently changed without matching repo evidence

## If Code Was Changed
- [ ] tests were run, or the reason they were not run is recorded
- [ ] related spec / plan / checklist docs were updated when needed
- [ ] handoff points to the exact file path or task id for the next session
